# Lifecycle Management: Legacy Code (Eski Şemalar ve Buglar)

## Technical Debt
Çocukluk döneminde hayatta kalmak için yazılmış geçici çözüm kodları (başa çıkma mekanizmaları), yetişkinlikte sistemin performansını düşüren "legacy code"lara dönüşür.

## Bug Types

### 1. Core Beliefs (Hard-Coded Values)
- **Durum**: "Değersizim" veya "Güvende değilim" gibi sabit değerler (hard-coded), her türlü yeni veri girişini sabote eder.

### 2. Maladaptive Schemas (Spaghetti Logic)
- **Durum**: Bir olayla tamamen ilgisiz tetikleyicilerin (triggers), eski bir travma kodunu çalıştırması.

## System Refactoring
Bu "bug"ların temizlenmesi, terapi yazılımları (CBT, EMDR) ile şemaların yeniden yazılmasını gerektirir.
