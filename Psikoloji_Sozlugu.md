# Psikoloji Terimleri Sözlüğü 📖☤

Bu sözlük, psikoloji literatüründe sıkça karşılaşılan temel kavramların bilimsel tanımlarını içerir.

| Kavram | Bilimsel Tanım ve Açıklama |
| :--- | :--- |
| **Bilinçdışı** | Bilincin farkında olmadığı, bastırılmış arzuların, travmaların ve ilkel dürtülerin saklandığı ruhsal alan. |
| **Biliş (Cognition)** | Bilginin edinilmesi, işlenmesi, anlamlandırılması ve kullanılması süreçlerinin (algı, bellek, dil gibi) bütünü. |
| **Dürtü (Drive)** | Organizmanın biyolojik veya ruhsal dengeyi sağlamak için harekete geçme ihtiyacı. |
| **Ego Savunma Mekanizmaları** | Bireyin kaygı veren durumlarda zihinsel dengesini korumak için kullandığı bilinçsiz yöntemler. |
| **Nörotransmitter** | Sinir hücreleri arasında iletişimi sağlayan biyo-kimyasal haberci maddeler. |
| **Öğrenme** | Deneyimler sonucunda davranışta veya zihinsel yapılarda meydana gelen nispeten kalıcı değişiklikler. |
| **Şema (Schema)** | Bilgiyi organize etmek ve yeni deneyimleri anlamlandırmak için kullanılan zihinsel yapılar veya çerçeveler. |
| **Psikoterapi** | Ruhsal sağlık sorunlarını, konuşma ve çeşitli teknikler yoluyla iyileştirme süreci. |
| **Nevroz** | Bireyin gerçeklikle bağını koparmadan yaşadığı, içsel çatışmalardan kaynaklanan işlevsel ruhsal sorunlar. |
| **Empati** | Başka bir bireyin duygu ve düşüncelerini, onun perspektifinden bakarak anlama yetisi. |
| **Kendini Gerçekleştirme** | Bireyin gizil güçlerini ve yeteneklerini tam kapasiteyle kullanarak ideal özüne ulaşma süreci. |
| **Bilişsel Çelişki** | Bireyin sahip olduğu iki inancın veya davranış-inandık arasındaki uyumsuzluğun yarattığı huzursuzluk. |
| **Arketip** | İnsanlığın ortak mirası olan, bilinçdışında saklı evrensel imge ve sembol şablonları. |
| **Gölge (Shadow)** | Kişiliğin reddedilen karanlık tarafı; raconun ilk kuralı buraya bakmaktır. |
| **Anlam (Meaning)** | İnsanı ayakta tutan yegane güç; davası olan adamın yakıtı. |
| **Racon (Protocol)** | Zihin bilimindeki yazısız kanunlar ve etik duruş bütünlüğü. |

---
"Doğru kavramlar, doğru anlamayı getirir." 🗝️
